# Synth API Documentation

## Overview

The Synth API provides endpoints to convert text and emotions into color palettes. All endpoints return JSON responses.

Base URL: `http://localhost:3001/api`

## Endpoints

### Health Check

**GET** `/api/health`

Returns the API health status.

**Response:**
```json
{
  "status": "healthy",
  "message": "Synth API - Synesthesia as a Service", 
  "version": "0.1.0"
}
```

### Text to Colors

**POST** `/api/synth/text-to-colors`

Convert text input to a color palette based on sentiment analysis and emotional keywords.

**Request Body:**
```json
{
  "text": "I feel creative and energetic!",
  "options": {
    "paletteSize": 5,
    "style": "harmonious",
    "intensity": "medium"
  }
}
```

**Parameters:**
- `text` (required): The input text to analyze
- `options.paletteSize`: Number of colors to generate (default: 5)
- `options.style`: Palette generation style ("harmonious", "complementary", "triadic")
- `options.intensity`: Color intensity ("low", "medium", "high")

**Response:**
```json
{
  "success": true,
  "input": "I feel creative and energetic!",
  "sentiment": {
    "score": 3,
    "comparative": 0.75,
    "tokens": ["feel", "creative", "and", "energetic"],
    "words": ["creative", "energetic"],
    "positive": ["creative", "energetic"],
    "negative": []
  },
  "palette": {
    "colors": ["#D4A574", "#4A7C7E", "#8E6BA8", "#A67C5A", "#7A9B7C"],
    "primary": "#D4A574",
    "emotions": ["creative"],
    "style": "harmonious",
    "intensity": "medium",
    "metadata": {
      "baseHue": 280,
      "baseSaturation": 60,
      "baseLightness": 70
    }
  },
  "metadata": {
    "timestamp": "2026-02-14T23:20:00.000Z",
    "engine": "synth-v0.1"
  }
}
```

### Export to CSS

**POST** `/api/synth/export/css`

Export a color palette as CSS custom properties.

**Request Body:**
```json
{
  "palette": ["#D4A574", "#4A7C7E", "#8E6BA8", "#A67C5A", "#7A9B7C"],
  "prefix": "synth"
}
```

**Response:**
```json
{
  "success": true,
  "format": "css",
  "output": ":root {\n  --synth-color-1: #D4A574;\n  --synth-color-2: #4A7C7E;\n  --synth-color-3: #8E6BA8;\n  --synth-color-4: #A67C5A;\n  --synth-color-5: #7A9B7C;\n}"
}
```

### Export to Figma

**POST** `/api/synth/export/figma`

Export a color palette in Figma-compatible format.

**Request Body:**
```json
{
  "palette": ["#D4A574", "#4A7C7E", "#8E6BA8", "#A67C5A", "#7A9B7C"]
}
```

**Response:**
```json
{
  "success": true,
  "format": "figma",
  "palette": [
    {
      "name": "synth-1",
      "color": "#D4A574",
      "description": "Generated by Synth from emotional input"
    }
  ],
  "instructions": "Import these colors directly into your Figma color styles"
}
```

## Color Theory

### Emotion Mapping

Synth uses both sentiment analysis scores and keyword detection to map emotions to colors:

| Emotion | Hue | Saturation | Lightness | Notes |
|---------|-----|------------|-----------|-------|
| Joy | 45° | 80% | 70% | Warm yellow |
| Excitement | 15° | 85% | 65% | Orange-red |
| Love | 345° | 70% | 75% | Pink-red |
| Peace | 200° | 40% | 80% | Light blue |
| Creative | 280° | 60% | 70% | Purple |
| Anger | 0° | 85% | 50% | Red |
| Sadness | 220° | 50% | 40% | Blue |

### Palette Generation

1. **Base Color**: Determined by emotional analysis
2. **Harmonious Colors**: Generated using color theory relationships
   - Complementary (180° hue shift)
   - Triadic (120° hue shift) 
   - Split-complementary (150° hue shift)
   - Analogous (30° incremental shifts)

## Error Responses

All errors return a JSON object with an `error` field:

```json
{
  "error": "Text input required",
  "example": {
    "text": "I feel creative and energetic!"
  }
}
```

## Rate Limits

Currently no rate limits are implemented. For production use, consider implementing rate limiting based on your usage patterns.

## Examples

### Basic Usage

```javascript
// Generate palette from emotion
const response = await fetch('/api/synth/text-to-colors', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    text: "I'm feeling nostalgic about summer days" 
  })
});

const data = await response.json();
console.log(data.palette.colors); // ["#E8B854", "#5E8B7E", ...]
```

### Export to Design Tools

```javascript
// Export as CSS variables
const cssResponse = await fetch('/api/synth/export/css', {
  method: 'POST', 
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    palette: data.palette.colors,
    prefix: 'theme'
  })
});

const css = await cssResponse.json();
console.log(css.output); // ":root { --theme-color-1: #E8B854; ... }"
```
