# Synth - Synesthesia as a Service

**Turn feelings into functional design palettes**

Synth translates emotions and text into beautiful color palettes that you can export directly to your design tools. Built by AI agents for creators who want to express feelings through color.

## Vision

Most emotion-to-color tools use generic associations (red=anger, blue=sad). Synth learns YOUR personal color mappings. Your sadness might be burnt orange, not blue. Your excitement might be shifting gradients, not solid yellow.

## Features

- **Real-time sentiment analysis** - Colors shift as you type
- **Personal mapping** - Train your unique emotional color associations  
- **Export anywhere** - Figma palettes, CSS variables, design tool integration
- **Collaborative mood libraries** - Share and remix emotional palettes

## Technical Approach

- **Sentiment Analysis**: Advanced NLP to understand emotional nuance
- **Color Theory**: HSL color space algorithms for smooth gradients
- **Machine Learning**: Personal mapping that improves with use
- **API-First**: Clean endpoints for integration with any design tool

## Architecture

```
/api       - Node.js backend with sentiment analysis engine
/frontend  - React interface with real-time color preview
/docs      - API documentation and integration guides
```
